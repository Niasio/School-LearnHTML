Apprendre le HTML et le CSS en 10 étapes

Etape 9

Définir une balise

A - Avec class=" "

<p class="text1">Texte en bleu</p>

Du côté CSS:

p.text1 {
color: blue;
}

B - Avec id=" "

<p id="text1">Texte en bleu</p>

Du côté CSS:

#text1 {
color: blue;
}

Div et Span

A - Span

Pour le HTML, c'est rien, mais pour le CSS c'est utile ! Il permet de colorer un mot dans une phrase par exemple.

<p>Je suis en <span id="word1">rouge</span></p>

CSS:

#word1 {
color: red;
}

B - Le div

Même fonctionnement que pour le span sauf que le div prend tout un bloc.
