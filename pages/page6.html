Etape 6

Le CSS

Cascading Style Sheets ou feuille de style en français. Il permet au site d'avoir un design plus beau.

Différence HTML/CSS

HTML:

<body bgcolor="#ffff00">

CSS:

body {
background-color: #ffff00;
}

Le CSS marche donc comme ceci:

selector {
property:value;
}

A - L'insérer

On inscrit dans le <head></head>

<html>
<head>
<title>Page test</title>
<link rel="stylesheet" type="text/css" href="style/style.css" />
</head>

[SUITE DU CODE]

</html>
