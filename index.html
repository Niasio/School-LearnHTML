Apprendre le HTML et le CSS en 10 étapes

Bienvenue sur notre site

Nous vous proposons d'apprendre à créer votre propre site comme celui-ci en seulement 10 étapes, 5 étapes pour le CSS et 5 autres pour le HTML.
Vous pouvez regardez notre code source et vous pouvez de même télécharger notre site via la plateforme GitHub (https://github.com/Niasio/SiteICN)
Nous sommes 2 élèves de Seconde au Saint-Esprit, et ce projet à été propulsé par le cours d'ICN.
Ce site contient 14 fichiers HTML, 1 fichier CSS et 13 images.
On a donc 85.4 % de code HTML et 14.6% de CSS.
Certaines étapes sont accompagnées d'exemple.

Nous vous souhaitons un bon apprentissage,
Bonne chance !
